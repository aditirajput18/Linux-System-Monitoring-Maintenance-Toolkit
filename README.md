# Linux-System-Monitoring-Maintenance-Toolkit
=======
# 🐧 Linux System Monitoring Toolkit

A collection of Bash scripts to monitor and maintain a Linux system. This toolkit provides real-time system information, process monitoring, disk usage analysis, network diagnostics, service status checks, backups, cleanup operations, logging, and automated health report generation.

---

## 📌 Features

- 📊 System Information Monitoring
- ⚙️ Process Monitoring
- 💾 Disk Usage Monitoring
- 🌐 Network Monitoring
- 🔍 Service Status Checking
- 👤 User Information
- 📦 Backup Automation
- 🧹 System Cleanup
- 📝 System Logging
- 📄 Health Report Generation
- ⏰ Cron Job Support

## 📜 Script Descriptions

| Script | Description |
|---------|-------------|
| **system_info.sh** | Displays hostname, current user, uptime, kernel version, and operating system information. |
| **process_monitor.sh** | Lists running processes and launches `htop` for real-time monitoring. |
| **disk_monitor.sh** | Shows disk space, directory usage, and warns when disk usage exceeds the defined threshold. |
| **network.sh** | Displays IP addresses, active network connections, and internet connectivity status. |
| **servicecheck.sh** | Checks whether essential Linux services are running. |
| **userinfo.sh** | Displays information about logged-in users and recent login history. |
| **backup.sh** | Creates compressed backups with timestamped filenames. |
| **cleanup.sh** | Removes package cache, unused packages, and old log files. |
| **logger.sh** | Stores monitoring information in `logs/system.log`. |
| **monitor.sh** | Generates a complete system health report and saves it to `reports/health-report.txt`. |

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/Linux-System-Monitor.git
```

Move into the project directory:

```bash
cd Linux-System-Monitor
```

Make all scripts executable:

```bash
chmod +x *.sh
```

---

## ▶️ Usage

Run any script individually.

Example:

```bash
./system_info.sh
```

Generate a health report:

```bash
./monitor.sh
```

Create a backup:

```bash
./backup.sh
```

Check running services:

```bash
./servicecheck.sh
```
